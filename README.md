# bundlewrap-centos-packages-base

`bundlewrap-centos-packages-base` simply installs some basic tools via `yum`.
Take a look into `items.py` for more details.

## Compatibility

This bundle has been tested on the following systems:

| OS          | `[x]` |
| ----------- | ----- |
| CentOS 7    | `[x]` |
| Fedora 24   | `[x]` |
| RHEL 7      | `[ ]` |
| Fedberry 23 | `[ ]` |
