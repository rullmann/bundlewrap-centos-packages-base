# bundlewrap-packages-base

`bundlewrap-packages-base` simply installs some basic tools via `dnf`.
Take a look into `items.py` for more details.

## Compatibility

This bundle has been tested on the following systems:

| OS          | `[x]` |
| ----------- | ----- |
| Fedora 24   | `[x]` |
| Fedberry 23 | `[ ]` |
